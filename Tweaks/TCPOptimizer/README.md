# TCP Optimizer
1. Download and open [TCPOptimizer.exe](https://github.com/moffa89/Apex-Legends-Autoexec-2025/raw/refs/heads/main/Tweaks/TCPOptimizer/TCPOptimizer.exe).
2. Import moffa89.spg and apply the Profile.
3. Download [Disable_BufferlistTracking.reg](https://github.com/moffa89/Apex-Legends-Autoexec-2025/blob/main/Tweaks/TCPOptimizer/Disable_BufferlistTracking.reg) file and open it.
4. Press Yes to apply.
5. If it doesn´t work rightclick and "open as Administrator".
6. Restart and Enjoy!

Note: If you want to revert the Settings you can apply default Settings in TCPOptimizer.
